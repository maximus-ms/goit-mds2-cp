# Дипломний Проєкт: Система інтелектуальної діагностики технічних пристроїв на основі акустичних сигналів з використанням методів машинного навчання

## Тема Дипломної Роботи

**Система інтелектуальної діагностики технічних пристроїв на основі акустичних сигналів з використанням методів машинного навчання**

## Опис Проєкту

Цей дипломний проєкт присвячений розробці інтелектуальної системи діагностики технічних пристроїв на базі мікроконтролера **ESP32-S3** та **4-мікрофонної решітки**. Система використовує методи машинного навчання та просторової обробки акустичних сигналів для виявлення нештатних ситуацій та діагностики стану промислових механізмів (двигунів, насосів, вентиляторів, генераторів).

### Ключові Особливості

- **Edge AI**: Виконання інференсу моделі безпосередньо на мікроконтролері без необхідності підключення до хмари
- **Few-Shot Learning**: Система навчається на мінімальній кількості прикладів нормальної роботи (близько 10 секунд запису)
- **Просторова фільтрація**: Використання мікрофонної решітки для покращення співвідношення сигнал/шум та придушення завад
- **Універсальність**: Агностична до типу обладнання — працює з будь-яким механізмом після короткого навчання
- **Низька вартість**: Загальна вартість компонентів становить ~$18-20

### Технологічний Стек

- **Апаратна платформа**: ESP32-S3 (Xtensa LX7 з векторними інструкціями)
- **Мікрофони**: 4× INMP441 (I2S MEMS мікрофони)
- **Framework**: ESP-IDF (C/C++)
- **DSP**: esp-dsp (оптимізовані FFT та векторні операції)
- **ML**: ESP-DL або TensorFlow Lite Micro
- **Алгоритм**: Siamese Network / One-Class Classification з косинусною відстанню

### Алгоритм Роботи

#### Етап 1: Навчання та Калібрування (Enrollment)
1. Запис ~10 секунд акустичних сигналів нормальної роботи технічного пристрою
2. Розбиття на вікна та обробка через Feature Extractor для виділення характеристик
3. Обчислення центроїдного вектора нормального стану пристрою
4. Збереження еталонного вектора в енергонезалежну пам'ять (NVS/Flash)

#### Етап 2: Діагностика та Моніторинг (Inference)
1. Запис коротких аудіо фрагментів від технічного пристрою
2. Препроцесінг акустичних сигналів (сумування сигналів з 4 мікрофонів, beamforming)
3. Обробка через ML модель для отримання вектора ознак
4. Порівняння з еталонним вектором через косинусну відстань
5. Діагностика стану пристрою та генерація сповіщення при виявленні відхилень

### Переваги Підходу

- ✅ Не потрібен великий розмічений датасет для кожного типу технічного пристрою
- ✅ Адаптивність до конкретного обладнання — система налаштовується під будь-який пристрій
- ✅ Працює в реальному або квазі-реальному часі для безперервної діагностики
- ✅ Покращення SNR на ~6 дБ завдяки решітці з 4 мікрофонів для точнішої діагностики
- ✅ Просторова селективність для придушення завад та виділення сигналу від діагностованого пристрою
